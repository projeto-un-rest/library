# Front-end of Library System

## Tech Stack

- [Vue.js 3](https://vuejs.org/)

## How to run this repository

### Clone this repository
```
git clone https://github.com/projeto-un-rest/library.git
```

### Access project folder
```
cd library
```

### Install dependencies
```
npm install
```

### Run project
```
npm run serve
```
